# Color discrimination asymmetry

Analysis and figure-generation code for a study of asymmetries in colour
discrimination, linking the chromatic statistics of natural scenes, human
psychophysical thresholds, MEG decoding, and deep neural network models.

Each script in the repository root regenerates one main or supplementary figure
from the aggregated data bundled under [`data/`](data/). Running a script writes
its output into [`figs/`](figs/).

## Requirements

- **MATLAB R2020a or later** (uses `exportgraphics`).
- **Statistics and Machine Learning Toolbox** (`groupsummary`, `unstack`,
  `ttest`, `pdist`, `linkage`, `cluster`).

No other toolboxes or third-party dependencies are required.

## How to run

1. Open MATLAB and set this repository as the current folder (or add it to the
   path). Scripts resolve all paths relative to their own location, so they can
   be run from any working directory.
2. Regenerate **all** figures at once:

   ```matlab
   run
   ```

   or run a single figure script, e.g.:

   ```matlab
   fig1_natural_objects_chromatic_distribution
   ```

   Each script adds [`utils/`](utils/) to the path automatically, loads the data
   it needs from [`data/`](data/), and saves figures into [`figs/`](figs/).

## Repository layout

```
.
├── run.m                        Regenerate every figure
├── fig1 … fig8, figS1 … figS9   Figure scripts (entry points)
├── utils/                       Shared helper functions and DKL colour-space utilities
├── data/                        Aggregated data needed to regenerate the figures
│   ├── object_pixels.mat        Natural-object pixels in DKL space (Fig. 1)
│   ├── meg_*.mat                MEG colour settings, decoding accuracies, log-odds ratios
│   ├── human/human_thresholds.csv   Human discrimination thresholds
│   ├── network/                 Per-network discrimination thresholds and Taskonomy results
│   └── dkl_hist_results/        Per-dataset DKL hue histograms and summaries
└── figs/                        Reference figure outputs (regenerated by the scripts)
```
